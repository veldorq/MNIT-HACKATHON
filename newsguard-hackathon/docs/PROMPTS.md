# AI Prompt Templates

## Backend Prompt
Create a Flask endpoint `/api/analyze` that accepts JSON `{ text }`, preprocesses text, predicts `real/fake`, computes credibility score (0-100), and returns confidence plus keyword breakdown with proper error handling.

## Frontend Prompt
Create a React component that includes a large textarea, loading state, disabled analyze button, and a results card with credibility score visualization, confidence bar, and flagged keyword chips.

## Deployment Prompt
Generate deployment steps for Vercel (frontend) and Railway/Render (backend), including environment variable setup and CORS-safe production config.
