# Market Analysis (Draft)

## Competitors
- Snopes
- FactCheck.org
- Full Fact
- PolitiFact
- Existing browser extensions with simple keyword checks

## Observed Gaps
- Limited free real-time text scoring APIs
- Weak explainability in most public tools
- Poor UX for non-technical users

## NewsGuard Positioning
- Real-time article-level analysis
- Credibility score with score breakdown
- Keyword and confidence transparency
- Hackathon-ready lightweight architecture

## Target Segments
- Students and educators
- Journalists and media teams
- Social-media-first users
- Civic fact-checking volunteers

## Demo Talking Points
- Explainability over black-box output
- Fast inference with practical UI
- Deployable architecture suitable for scale-up
